Fatebringer

Genre:

2D Action/Shooter scroller based on Destiny, a game created by Bungie.

Gameplay:

Basic "W-A-S-D" keyboard movement, "SPACEBAR" to jump, "Alpha NUMs" for special abilities & mouse controls.

Description:

Fatebringer is a 2D scroller with 5 different locations to play in. The character has 5 different abilities with basic keyboard & mouse controls. The mouse moves the arms of the character and that is how you aim the gun.

Each location has its own enemy types and final boss. Some enemies are stronger than others and take more bullets to be defeated or deal more damage to the player.

Requirements:

No specific requirements. Originally intended to be an OpenGL build but it exceeded the allowed file size.

Disclaimer:

I do NOT own anything related to Destiny or have any relationship to Bungie. This project was created for learning purposes.

Savior

Genre:

2D Survival/Slasher

Gameplay:

Basic "W-A-S-D" keyboard movement, "SPACEBAR" to jump, mouse controls for attack & "P" for pause.

Description:

Savior is a 2D Survival/Slahser with multiple levels to play in. Besides the normal mode, there is also a hard mode for those wanting a challenge.

Each location has its own enemy types and timers for staying alive. Some enemies are stronger than others and will behave slightly differently.

Requirements:

No specific requirements. Originally intended to be an OpenGL build but it exceeded the allowed file size.

Grumpy

Genre:

2D Physics Game.

Gameplay:

Simple mouse click + drag controls.

Description:

Grumpy is a simple 2D game created for learning purposes.

The goal of the game is to hit the enemies and make points, hitting crates also create points.

Requirements:

It is an OpenGL build, make sure to have a browser that supports it.
